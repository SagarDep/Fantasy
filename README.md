# FantasyMusicPlayer

**正在无限期开发中... 尽请期待

**特性**
- 无广告, 快速且顺滑
- 支持歌词显示和自动滚动
- 支持通过滑动歌词来调整播放进度
- 支持本地/在线歌词(数据来自 http://tingapi.ting.baidu.com/v1/restserver/ting)
- 优先通过ID3v2从歌曲文件本身获取歌曲信息(例如封面,歌词等),最大限度的节省网络开销
- 支持定时停止播放, 并支持在当前播放的音乐结束后再停止播放
- 支持按照时间和文件路径来过滤歌曲.
